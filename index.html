About me

I’m a frontend developer with expertise in HTML, CSS, JavaScript, React, and Tailwind, focused on creating responsive and user-friendly web experiences. My projects reflect a strong commitment to quality and innovation, with a standout capstone project that ranked in the top 20 out of 120 groups.


I’m passionate about turning ideas into seamless interfaces that deliver exceptional user experiences. I’m always ready to tackle new challenges and collaborate on projects that make a real impact.

projects

Project 1

Emirate booking website cloned

The Emirates website clone was an ambitious project where I aimed to replicate the intricate design and functionality of a well-established airline website. This task required a strong focus on layout, responsiveness, and interactive elements to ensure that the user experience mirrored the original. Through this project, I honed my skills in frontend technologies, including HTML, CSS, and JavaScript, and gained valuable experience in tackling complex design challenges. The process of cloning such a sophisticated website deepened my understanding of creating user-centric, high-quality web interfaces.

Project 2

Medtrust

I developed a website for medicine authentication that empowers users to verify their drugs using either a scanner or a unique code. Additionally, the website features a tool to locate the nearest pharmacy, ensuring easy access to genuine medications. This project was focused on enhancing user safety and convenience, combining advanced technology with a user-friendly interface.

Project 3

sellsmart

Developed the “Sell Smart” website, a comprehensive POS system designed with distinct roles for managers, admins, and cashiers. This platform streamlines sales processes, inventory management, and financial tracking, making it an essential tool for efficient business operations. The focus on role-specific functionalities ensures that each user has the tools they need to perform their tasks effectively, all within a secure and user-friendly interface.

Project 4

An interior website

I developed an elegant interior design website that combines style with functionality. The site features a robust search functionality, allowing users to easily find their perfect pieces. With a focus on ease of navigation and a fully responsive design, it ensures a seamless experience across all devices. To top it off, multiple payment options are integrated, making the purchasing process as smooth and enjoyable as browsing the site itself. This project was all about creating a beautiful and user-friendly online space for interior enthusiasts.
