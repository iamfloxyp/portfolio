projects

This task required a strong focus on layout, responsiveness, and interactive elements to ensure that the user experience mirrored the original. Through this project, I honed my skills in frontend technologies, including HTML, CSS, and JavaScript, and gained valuable experience in tackling complex design challenges. I developed a website for medicine authentication that empowers users to verify their drugs using either a scanner or a unique code. Additionally, the website features a tool to locate the nearest pharmacy, ensuring easy access to genuine medications. This project was focused on enhancing user safety and convenience, combining advanced technology with a user-friendly interface.
